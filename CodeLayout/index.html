Chào Mừng Đến Với Website Demo

Đây là trang chủ của website. Nội dung sẽ được thêm vào đây.

Về Website Này

Website này được tạo ra như một ví dụ đơn giản về cách xây dựng layout với menu và footer. Học viên có thể sử dụng mã nguồn này để học và phát triển thêm.

Đặc biệt: Menu và Footer được tách ra thành file riêng (header.html và footer.html) để code ngắn gọn và dễ quản lý hơn!

Tính Năng

  • Menu đơn giản và dễ sử dụng
  • Responsive design (tương thích với mobile)
  • Footer với thông tin liên hệ
  • Code rõ ràng, dễ hiểu
  • Layout được tách riêng để tái sử dụng

Cách Layout Hoạt Động

Header và Footer được lưu trong file riêng:

  • header.html - Chứa menu điều hướng
  • footer.html - Chứa thông tin footer

JavaScript sẽ tự động load 2 file này vào mỗi trang. Như vậy khi bạn muốn sửa menu hoặc footer, chỉ cần sửa 1 file là đủ!
