Giới Thiệu Về Chúng Tôi

Chào mừng bạn đến với trang giới thiệu của chúng tôi.

Về Dự Án

Đây là một dự án mẫu được tạo ra để giúp các học viên mới học HTML, CSS và JavaScript có thể hiểu và áp dụng các khái niệm cơ bản.

Layout này bao gồm một menu điều hướng (navigation bar) ở trên cùng và một footer ở dưới cùng trang. Menu và footer sẽ xuất hiện trên tất cả các trang của website.

Mục Tiêu

  • Cung cấp một template đơn giản và dễ hiểu
  • Giúp học viên học cách tạo layout cơ bản
  • Hướng dẫn cách làm responsive design
  • Code rõ ràng với nhiều comment giải thích
  • Học cách tách layout để tái sử dụng code

Công Nghệ Sử Dụng

HTML5: Cấu trúc nội dung của website

CSS3: Tạo kiểu dáng và bố cục cho website

JavaScript: Thêm tính tương tác và load layout động

Fetch API: Load nội dung từ file header.html và footer.html
